# 바른스트레칭 홈페이지

국가대표 물리치료사가 개발한 프리미엄 패시브 스트레칭 서비스 소개 페이지.

## 구성

- `index.html` — 전체 소개 페이지 (싱글 페이지, 앵커 네비게이션)
- `assets/instructors/` — 강사 프로필 이미지 (PNG)

## 섹션

1. **센터 방문 (B2C)** — 4종 프로그램 + 요금표
2. **기업 방문 (B2B 출장)** — 이메일 문의 기반 (가격 비공개)
3. **아카데미** — 원데이 · 본과정 / 일정은 인스타그램 공지

## 강사

- 대표 · 송치웅
- 서울숲점 지점장 · 김수현
- 팀장 · 오승은
- 시니어코치 · 권승준 *(사진 수령 대기)*
